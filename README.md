# Password Manager - By_ MRZERO

## Deskripsi
Password Manager adalah aplikasi sederhana untuk menyimpan dan mengelola akun pengguna dengan aman menggunakan enkripsi. Aplikasi ini dilengkapi dengan fitur backup otomatis, sinkronisasi cloud, dan antarmuka GUI.

**Pengembang:** MRZERO  
**Lisensi:** Hak cipta dilindungi. Dilarang menduplikasi atau mengubah kode tanpa izin.

## Fitur Utama
- Penyimpanan akun terenkripsi.
- Backup otomatis dan manual.
- Sinkronisasi data ke cloud.
- Antarmuka GUI untuk kemudahan penggunaan.

## Persyaratan
- Python 3.10+
- Library: `tkinter`, `schedule`, `shutil`, `logging`
